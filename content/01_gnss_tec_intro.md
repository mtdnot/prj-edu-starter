# GNSSで電離圏TECはどう出す？（L0→L3）

> ※このページは `status: draft` のため、公開フィードには載りません。

## L0：完全一般

衛星から届く電波は、空の「見えない層」（電離圏）を通ると、ほんの少し遅れます。
その遅れ方を上手く見積もると、電離圏の状態（電子の多さ）を推定できます。

## L1：中学〜高校

- 電波は速さがほぼ一定（真空中の光速）
- でも媒質があると“遅れ”が出る
- GNSSは周波数の違う電波を複数使う
- 周波数で遅れ方が違うので、差を取ると電離圏成分が浮き出る

## L2：大学理系

ここで扱うのは「観測方程式」「バイアス」「ノイズ」です。

- 擬似距離・搬送波位相の観測モデル
- 1次近似で電離圏遅延は周波数の二乗に反比例
- 二周波線形結合からSTECを推定
- 受信機・衛星のDCB（Differential Code Bias）をどう扱うか

## L3：実装（一次資料）

- 入力：RINEX（観測ファイル）
- 出力：STEC/VTEC（時系列）
- 手順：
  1. RINEX読み込み
  2. 欠損/外れ値処理
  3. 二周波結合
  4. DCB補正
  5. VTEC変換（仮定の明示）
